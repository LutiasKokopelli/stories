    Learn When to Quit Writing Bonus Chapters Because
    Nobody Will Read Them

    Welcome to LWtQ’s side corner of mini stories and cut content! Every snippet of story told in this section is entirely canon, and has only been kept in the dark so far due to length and plot restrictions. If you wanted to explore all the dark corners and perspectives the original story didn’t have time to tackle, here they are!

    For the sake of clarity, each excerpt will be associated with various tags in order to clarify when exactly in the story’s timeline they take place.

    The Tags

    First comes the chapter tag. Each title for the chapters will start with a 3-digit number in the tags, which indicates the number of the chapter in the main fanfic you should have read up to before reading the corresponding chapter. If you see the tag [FIC], it means that this excerpt contains spoilers towards the entirety of the story and that you should read it only once the main fanfic is complete. If you see [000], it means that the chapter shouldn’t contain any spoilers regarding the main LWtQ story and that you should be able to read them without having even started reading the main fanfic... Not that I’d recommend that to happen, but you do you.

    Second, you will see either [UT] or [DU]: the first one means that the chapter is centered around the Undertale universe, the second that it will be set in Dawn’s original universe. Naturally, the former should appear more frequently than the latter.

    Thirdly, the tag will be either [FIC] or [ART]: the first one means that it is a written chapter, the second one that it is drawn.

    Finally, if you see the tag [NOT CANON], you know what to expect. These chapters should exclusively be random funny comics that I would’ve drawn just for the sake of telling a joke or drawing the characters in silly situations.

    It is also possible that, if multiple updates are related to a single theme or storyline/overarching “plot” (besides the obvious LWtQ plot itself), one last tag will be present. This last tag will probably be in the form of a single emoji that all chapters related to that theme/story arc will possess. Since it is possible that some story arcs will contain major spoilers, their significance will only be revealed inside the description of the chapters themselves.
